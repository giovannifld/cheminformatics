# Cheminformatics
## Molecule Analyzer
## This Google Colab notebook provides a simple tool to analyze chemical molecules based on their SMILES (Simplified Molecular-Input Line-##Entry System) string. It utilizes the rdkit library to calculate key molecular properties and visualize the molecular structure.

## Features
## Molecular Weight (MW): Calculates the molecular weight of the input molecule.
## LogP (hydrophobicity proxy): Estimates the octanol-water partition coefficient, a measure of hydrophobicity.
## TPSA (Topological Polar Surface Area): Calculates the topological polar surface area, which is related to drug absorption.
## Molecular Image: Generates and displays a 2D image of the molecule.
## How to Use
## Run the code cell: Execute the Python code cell in this notebook.
## Enter SMILES string: When prompted, enter a valid SMILES string for the molecule you wish to analyze (e.g., CCO for Ethanol, CC(=O)Oc1ccccc1C(=O)O for Aspirin).
## View Results: The notebook will then output the calculated molecular weight, LogP, TPSA, and a 2D image of the molecule.
## Dependencies
## This notebook automatically installs the rdkit library, which is essential for cheminformatics functionalities.
